18 年多前，Spring 带着让 Java 应用程序开发更简单的基本使命进入了开发领域。最初，这意味着为 EJB 2.x 提供一个轻量级的替代方案。多年来，Spring 不断扩展其简单性使命，以应对常见的开发挑战，包括持久性、安全性、集成、云计算等。
